Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets
  • Used to add styling to the elements created with HTML
  • Determines how elements are presented on the page, including font, color, size, positioning of elements, and even animation

Git

  • A version control system that stores your code and its history, as well as separate versions of the code
  • Commands
  • A system that allows you to create "save points" (called commits) of your work. It's good practice to commit your work whenever you reach a good stopping point. This creates a history of changes and allows you to revert to an earlier version if necessary.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A lightweight programming language that is used to provide logic and interactivity for websites
